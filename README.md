श्रीरामचंद्र कृपालु भजमन, हरण भवभय दारुणं
नव कंज लोचन कंज मुख, कर कंज पद कंजारुणं
कन्दर्प अगणित अमित छवि नव नील नीरज सुंदरम्

भजु दीनबन्धु दिनेश दानव, दैत्य वंश निकन्दनं
रघुनन्द आनन्द कन्द कोशल, चन्द दशरथ नन्दनम्
सिर मुकुट कुण्डल तिलक, चारु धारु अङ्ग विभूषणं

मन चाहि राज्यौ मिलहि सोबर, सहज सुन्दर साँवरौ
